# Transformice Keys
An API parser to get the Transformice connection keys.

### Usage
To use this program you'll have to obtain an API key [here](https://github.com/Tocutoeltuco/transfromage/blob/master/README.md#keys-endpoint), then enter the keys in `api.json`.

If you choose to save the connection keys they will be saved in `keys.json` and `keys.txt`.
